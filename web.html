Code Examples

Service Setup & VNic Configuration
// Create system configuration
config := &types.SysConfig{
    LocalUuid: "node-123",
    VnetPort: 8080,
    MaxDataSize: 1024*1024,
    TxQueueSize: 1000,
    RxQueueSize: 1000,
}

// Add services to configuration
ifs.AddService(config, "user-service", 1)
ifs.AddService(config, "auth-service", 2)

// Create and start VNic
vnic := NewVNic(config, resources)
vnic.Start()
Communication Patterns
// Unicast to specific destination
err := vnic.Unicast("node-456", "user-service", 
    1, ifs.POST, userData)

// Round-robin to service providers
destination, err := vnic.RoundRobin("auth-service", 
    2, ifs.GET, request)

// Request with response
response := vnic.Request("node-789", 
    "data-service", 1, ifs.GET, query)

// Multicast to all providers
err = vnic.Multicast("notification-service", 
    1, ifs.Notify, event)

Key Components

Query Engine

Expression-based querying with comparators, conditions, sorting, pagination, and schema filtering for complex data retrieval.

Type Registry

Dynamic type registration and introspection with RNode reflection system, decorators, and table views for runtime flexibility.

Notification System

Property change notifications with batching, sequencing, and event propagation for real-time system updates.

Network Modes

Support for Native, Docker, and Kubernetes networking environments with automatic adaptation and configuration.
